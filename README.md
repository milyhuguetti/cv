# CV Web- Milagros Huguetti

Este repositorio es un cv tecnico desarrollado en formato  (HTML y CSS).


## Descripcion
Se trata de una pagina web simple donde presento mi perfil profesional, tecnologias y proyectos.

##  Tecnologías utilizadas
- HTML
- CSS
- Responsive Design


## Funcionalidades
Presenta:
- presentacion de perfil profesional.
- Lista de tecnologias.
- Proyectos personales con enlace.
-Informacion de contacto.

Hecho con **HTML y CSS**, responsive para dispositivos móviles.
